# DigiCam Dreams ✦

a vintage digicam-style photo filter web app — cute, aesthetic, and fully in-browser.

![digicam dreams](https://img.shields.io/badge/aesthetic-vintage%20digicam-f4b8c8?style=flat-square) ![HTML](https://img.shields.io/badge/built%20with-HTML%20%2B%20Canvas-d4708a?style=flat-square)

## ✨ features
<img width="1362" height="779" alt="image" src="https://github.com/user-attachments/assets/23780235-7f90-442b-b75a-0aa3a921d1e0" />
<img width="1355" height="706" alt="image" src="https://github.com/user-attachments/assets/d013c893-ff3e-42b3-98b5-13863e5b7e9b" />

- 🎞️ multiple vintage filter presets (y2k, lomography, polaroid & more)
- 🎛️ adjustable sliders — brightness, contrast, saturation, grain, vignette
- 🖼️ overlay effects for that real digicam feel
- 💾 download your edited photo directly
- 🌸 fully client-side — no uploads, no backend, just vibes

## 🛠️ tech used

- HTML5 Canvas API — all filters applied via pixel manipulation
- Vanilla JS — no frameworks, no dependencies
- CSS3 — maroon + rose aesthetic with custom variables
