# DigiCam Dreams ✦

a vintage digicam-style photo filter web app — cute, aesthetic, and fully in-browser.

- 🎞️ multiple vintage filter presets (y2k, lomography, polaroid & more)
- 🎛️ adjustable sliders — brightness, contrast, saturation, grain, vignette
- 🖼️ overlay effects for that real digicam feel
- 💾 download your edited photo directly
- 🌸 fully client-side — no uploads, no backend, just vibes
